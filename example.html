Simple Cookie Consent

This page is an example of a simple Cookie Consent Management pop-up built using JavaScript to work in alongside Google Tag Manager (GTM).

Don't see the pop-up? You may have already accepted/declined it. Use the button below to trigger it again.

Test This Page

This page is firing three Custom HTML tags via GTM, which log a message in the developer console.

  1. One tag fires on all page views.
  2. One tag fires only on page views with cookies allowed. (This tag also fires when fresh consent is given)
